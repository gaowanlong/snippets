Kernel development process
==========================

## Release big picture

v4.0-->v4.1-rc1...-->v4.1-rc8-->v4.2

## Patch life cycle

- Design
- early review -- mail list
- wider review -- -next tree
- rebase and rebase again -- low priority
- go into mainline
- stable release - if critical
- Longterm maintain
